# ios_ShopManagement

Приложение для отправки чеков через WhatsApp, данные используются из Google таблицы, которая должна иметь следующий формат:
1. Наименование	
2. Цена продажи	
3. Ссылка на чек	
4. Фото	
5. Описание

Для просмотра у Google аккаунта должен быть доступ к просмотру таблицы.
